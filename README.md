# OloTune releases

![OloTune](screenshot.png)

Windows installers for [OloTune](https://github.com/olopad-com/olotune) —
a free guitar tuner VST3 + standalone, built by [OloPad](https://olopad.com).

Pick the latest installer from the
[Releases](https://github.com/olopad-com/olotune-releases/releases/latest)
page.

## Install notes

- Windows 10 / 11, x64.
- Requires the WebView2 runtime (pre-installed on Windows 11; Windows 10
  users can grab it from
  [Microsoft](https://developer.microsoft.com/microsoft-edge/webview2/)).
- The installer offers a per-user or per-machine VST3 location; pick
  whichever your DAW scans.
